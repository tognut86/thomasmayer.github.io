Countless Lives

A living collage

Countless Lives is a living collage, a work in progress. The layers accumulate, change in time, shaping the present image. â€‹
Combining projection, video cameras, music, and his own body, the performer enacts a live recording loop of a slow, expansive movement sequence to create a video collage. The audience is invited to view the process, contributing with visual input via live-stream. Each of the elements in the performance space is captured in the video, creating a unique visual image which will, in turn, shape future layers of the collage.
It is an immersive experience; a collaborative work which evolves with every iteration.
